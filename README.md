## 前言

随着移动互联网的迅猛发展，小程序因其无需下载、即点即用的便捷性，成为了企业产品推广的新渠道。本项目是基于小程序的企业产品推广系统，后端采用PHP进行开发，以Java为主要编程语言，整合了Spring、Springmvc、MyBatis等主流框架，旨在为广大企业提供一套高效、易用的产品推广解决方案。

## 内容介绍

本项目主要包括以下功能模块：产品展示、产品分类、产品搜索、用户评论、购物车等。通过微信小程序前端，用户可以轻松查看企业产品信息，实现一键分享、快速传播。后端采用PHP编写，确保系统稳定性和安全性。此外，项目还支持Uniapp前端技术，实现多端适配，让您的产品覆盖更广泛的目标用户。

## 技术介绍

- 语言：Java
- 使用框架：Spring、Springmvc，MyBatis，微信小程序
- 前端技术：JS、Vue、CSS3，Uniapp
- 开发工具：IDEA/Eclipse，Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
